Общее описание
Проект представляет собой многофункциональное консольное приложение на Java, включающее системы для управления студентами и их оценками, телефонный справочник с поиском и управлением контактами, геометрическую систему с различными фигурами и вычислениями, а также систему валидации ввода пользователя.

Структура проекта
Пакеты:
ru.masyutin.main - главный класс приложения

ru.masyutin.education - классы для работы со студентами

ru.masyutin.contacts - телефонный справочник

ru.masyutin.geometry - геометрические фигуры 

ru.masyutin.utils - утилиты валидации ввода

Детальное описание классов
Geometry Package
Shape (абстрактный класс)
Назначение: Базовый класс для всех геометрических фигур.

Поля:

name: String - название фигуры

Методы:

getArea(): double - абстрактный метод вычисления площади

getInfo(): String - возвращает информацию о фигуре с площадью

toString(): String - абстрактный метод строкового представления

Point
Назначение: Представляет точку в 2D-пространстве.

Поля:

x: int, y: int - координаты точки

Методы:

distanceTo(Point other): double - расстояние до другой точки

distanceToOrigin(): double - расстояние до начала координат

equals(Object obj): boolean - сравнение точек по координатам

clone(): Point - создание копии точки

Point3d
Назначение: Наследник Point, представляет точку в 3D-пространстве.

Дополнительные поля:

z: int - координата Z

Методы:

Переопределяет toString() для вывода 3D-координат

Circle
Назначение: Представляет круг.

Поля:

center: Point - центр круга

radius: int - радиус

Методы:

getArea(): double - площадь круга по формуле pi*r^2

Валидация радиуса (должен быть больше 0)

Rectangle
Назначение: Представляет прямоугольник.

Поля:

topLeft: Point - левая верхняя точка

width: int, height: int - размеры

Методы:

getArea(): double - площадь = width × height

Валидация ширины и высоты (должны быть больше 0)

Square
Назначение: Наследник Rectangle, представляет квадрат.

Поля:

topLeft: Point - левая верхняя точка

side: int - длина стороны

Уникальные методы:

getPolyline(): List<Point> - возвращает замкнутую ломаную линию квадрата

getPerimeter(): double - периметр = 4 × side

Triangle
Назначение: Представляет треугольник.

Поля:

pointA, pointB, pointC: Point - вершины треугольника

Методы:

getArea(): double - вычисляет площадь по формуле Герона

ShapeSum
Назначение: Вспомогательные методы для работы с коллекциями фигур.

Методы:

calculateTotalArea(List<Shape>): double - суммарная площадь всех фигур

printShapesInfo(List<Shape>) - вывод информации о фигурах

Education Package
Student
Назначение: Представляет студента с оценками.

Поля:

name: String, group: String - имя и группа

grades: List<Integer> - список оценок

Основные методы:

addGrade(int grade) - добавление оценки с валидацией (2-5)

getAverageGrade(): double - средний балл

isExcellentStudent(): boolean - проверка, все ли оценки равны 5

hasUnsatisfactoryGrades(): boolean - есть ли оценки 2

countGrade(int target): int - подсчет конкретных оценок

InvalidGradeException
Назначение: Исключение для невалидных оценок.

Contacts Package
PhoneBook
Назначение: Телефонный справочник с двусторонним поиском.

Поля:

contacts: Map<String, String> - телефон к имени

nameToPhone: Map<String, String> - имя к телефону

Основные методы:

addContact(phone, name): String - добавление/обновление контакта

removeContactByName(name): boolean - удаление по имени

getPhoneByName(name): String - поиск телефона по имени

getNamesByPrefix(prefix): String[] - поиск по началу имени

Utils Package
InputValidator
Назначение: Валидация пользовательского ввода.

Основные методы валидации:

getValidInt() - целое число

getValidStudentName() - имя без цифр

getValidPhone() - телефон (только цифры и +)

getValidGrade() - оценка 2-5

isValidTriangle() - проверка неколлинеарности точек

Алгоритмы выполнения
Алгоритм вычисления расстояния между точками (Point.distanceTo)
Вычисляется разность координат по оси X: dx = this.x - other.x

Вычисляется разность координат по оси Y: dy = this.y - other.y

Вычисляется квадрат расстояния: dx * dx + dy * dy

Возвращается квадратный корень из полученной суммы

Алгоритм вычисления площади треугольника (Triangle.getArea)
Вычисляются длины всех трех сторон треугольника:

a = расстояние между точками A и B

b = расстояние между точками B и C

c = расстояние между точками C и A

Вычисляется полупериметр: p = (a + b + c) / 2

Применяется формула Герона: площадь = sqrt(p × (p - a) × (p - b) × (p - c))

Алгоритм создания замкнутой ломаной линии квадрата (Square.getPolyline)
Вычисляются координаты всех четырех вершин квадрата:

Левая верхняя: (x1, y1) = исходная точка topLeft

Правая верхняя: (x2, y2) = (x1 + side, y1)

Правая нижняя: (x3, y3) = (x2, y1 + side)

Левая нижняя: (x4, y4) = (x1, y3)

Точки добавляются в список в порядке обхода по часовой стрелке

Первая точка добавляется в конец списка для замыкания ломаной

Алгоритм добавления контакта в телефонную книгу (PhoneBook.addContact)
Проверяется существование имени в nameToPhone мапе

Если имя существует, извлекается старый телефон и удаляется из contacts мапы

Проверяется существование телефона в contacts мапе

Если телефон существует, извлекается старое имя и удаляется из nameToPhone мапы

Добавляется новая пара телефон-имя в contacts мапу

Добавляется новая пара имя-телефон в nameToPhone мапу

Возвращается старый телефон (если был)

Алгоритм поиска имен по префиксу (PhoneBook.getNamesByPrefix)
Создается пустой список для подходящих имен

Для каждого имени в keySet мапы nameToPhone:

Проверяется, начинается ли имя с заданного префикса

Если начинается, имя добавляется в результирующий список

Список преобразуется в массив и возвращается

Алгоритм вычисления среднего балла студента (Student.getAverageGrade)
Если список оценок пуст, возвращается 0.0

Вычисляется сумма всех оценок в списке

Сумма делится на количество оценок

Возвращается результат как число с плавающей точкой

Алгоритм проверки треугольника на валидность (InputValidator.isValidTriangle)
Вычисляется площадь треугольника по координатам вершин:
area = A.x × (B.y - C.y) + B.x × (C.y - A.y) + C.x × (A.y - B.y)

Если площадь равна 0, точки лежат на одной прямой (треугольник невалиден)

Если площадь не равна 0, точки образуют валидный треугольник

Алгоритм валидации имени студента (InputValidator.getValidStudentName)
Запрашивается ввод строки

Проверяется минимальная длина (не менее 2 символов)

Проверяется каждый символ строки:

Если символ является цифрой, ввод отклоняется

Если символ не является буквой, пробелом или дефисом, ввод отклоняется

Проверяется наличие хотя бы одной буквы в строке

Если все проверки пройдены, имя возвращается

Алгоритм возведения в степень (Main.powerFromStrings)
Парсятся строковые аргументы xStr и yStr в целые числа

Вычисляется x в степени y с помощью Math.pow()

Результат возвращается с округлением до 2 десятичных знаков

Алгоритм главного меню (Main.main)
Проверяются аргументы командной строки

Если передано 2 аргумента, выполняется возведение в степень

Если аргументов нет, запрашивается ввод с клавиатуры

Отображается главное меню с 7 опциями

В зависимости от выбора пользователя запускается соответствующая задача

Цикл продолжается до выбора опции "Выход"

Алгоритм работы со студентами (Main.runStudentTask)
Запрашивается имя и группа студента с валидацией

Создается объект Student

Отображается подменю действий:

Добавление оценки с валидацией (2-5)

Отображение информации о студенте

Отображение статистики (средний балл, количество оценок, отличник и т.д.)

После завершения работы возврат в главное меню

Алгоритм работы с телефонной книгой (Main.runPhoneBookTask)
Создается пустой объект PhoneBook

Отображается подменю действий:

Добавление контакта с валидацией телефона и имени

Поиск телефона по имени

Удаление контакта по имени

Отображение всех контактов

Поиск имен по начальной части

Отображение статистики справочника

Все операции используют двусторонние мапы для эффективного поиска

Алгоритм демонстрации полиморфизма (Main.demonstratePolymorphism)
Создается коллекция объектов типа Shape

Пользователю предлагается создать различные фигуры (круг, квадрат, прямоугольник, треугольник)

Для каждой созданной фигуры вычисляется площадь через метод getArea()

Выводится информация о всех фигурах в коллекции

Вычисляется и отображается общая площадь всех фигур

Демонстрируется работа виртуальных вызовов метода getArea()

Алгоритм клонирования точек (Main.demonstratePointCloning)
Создается оригинальная точка

Вызывается метод clone() для создания копии

Проверяется, что оригинал и клон являются разными объектами

Проверяется, что оригинал и клон имеют одинаковые координаты

Изменяются координаты оригинала

Демонстрируется, что клон сохраняет исходные значения


Проект демонстрирует принципы объектно-ориентированного программирования, включая наследование, полиморфизм, инкапсуляцию и обработку исключений, а также эффективные алгоритмы для работы с различными структурами данных.
